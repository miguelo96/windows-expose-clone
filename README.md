A macOS Expos√©-style window switcher for Windows 11.

## Features
- Press **Win+W** to scatter all open windows across your screen
- Click any window to switch to it
- Press **Escape** to cancel
- Smooth animations with blurred wallpaper background
- Organic window placement algorithm (no overlaps!)

## Requirements
- Windows 11 (or Windows 10)
- .NET Framework 8+

## Installation
1. Download the latest release
2. Run `WindowScatter.exe`
3. Add to startup folder for Win+W hotkey on boot

## Building from Source
1. Open in Visual Studio 2019+
2. Build solution
3. Run
